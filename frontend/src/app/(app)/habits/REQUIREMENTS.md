# 해빗 트래커 요구사항

## 개요

아날로그 플래너의 습관 추적 기능을 디지털로 구현한 페이지.
사용자가 습관을 직접 관리하고, 월간 단위로 달성 현황을 확인할 수 있다.

---

## 기능 요구사항

### 1. 습관 관리 (CRUD)

- 사용자가 습관을 직접 추가/수정/삭제할 수 있다
- 습관 생성 시 아래 속성을 정의한다:

| 속성     | 설명                                           | 필수 여부 |
| -------- | ---------------------------------------------- | --------- |
| 이름     | 습관명                                         | 필수      |
| 카테고리 | 사용자 정의 카테고리 (예: 건강, 학습, 생활 등) | 선택      |
| 색상     | 습관 카드/뱃지 색상                            | 선택      |
| 아이콘   | 습관을 나타내는 아이콘                         | 선택      |

### 2. 3단계 목표 시스템

습관을 생성할 때 3단계 목표를 정의한다. 그날의 컨디션에 따라 달성할 단계를 선택할 수 있다.

| 단계           | 설명                                     |
| -------------- | ---------------------------------------- |
| 1단계 (쉬움)   | 힘든 날에도 최소한 지킬 수 있는 목표     |
| 2단계 (보통)   | 평소 컨디션에서 달성하는 기본 목표       |
| 3단계 (챌린지) | 컨디션이 좋은 날 도전하는 최고 수준 목표 |

**예시 - "독서" 습관:**

- 1단계: 5분 읽기
- 2단계: 30분 읽기
- 3단계: 1시간 읽기

### 3. 체크 방식

- 각 날짜별로 체크박스 형태로 달성 여부를 기록한다
- 달성 시 3단계 중 어느 단계를 완료했는지 선택하여 기록한다
  - 미완료: 체크 없음
  - 1단계 완료: 1단계 표시
  - 2단계 완료: 2단계 표시
  - 3단계 완료: 3단계 표시

### 4. 습관 통계

각 습관별로 아래 통계를 표시한다:

| 항목        | 설명                              |
| ----------- | --------------------------------- |
| 현재 연속일 | 오늘 기준으로 연속 달성 중인 일수 |
| 최대 연속일 | 역대 가장 길게 연속 달성한 일수   |
| 누적일      | 전체 기간 동안 달성한 총 일수     |

- 연속일 계산 기준: 1단계 이상 달성 시 연속으로 간주

### 5. 뷰 형식

#### 현재 구현 범위: 월간 뷰

습관별로 카드 섹션으로 표시한다.

**카드 구성:**

- 습관명 (아이콘, 색상 포함)
- 통계: 현재 연속일 / 최대 연속일 / 누적일
- 월별 캘린더: 해당 월의 날짜별 달성 단계 표시

**인터랙션:**

- 캘린더의 각 날짜 셀을 클릭하면 ContextMenu가 표시된다
- ContextMenu 선택지:
  - 1단계 달성 (쉬움)
  - 2단계 달성 (보통)
  - 3단계 달성 (챌린지)
  - 선택 제거 (이미 달성 기록이 있는 경우에만 표시)

```
┌─────────────────────────────────────────┐
│ 📚 독서                                  │
│ 현재 연속 3일 · 최대 10일 · 누적 24일    │
│                                         │
│  일  월  화  수  목  금  토             │
│   1   2   3   4   5   6   7            │
│  [3] [2] [ ] [1] [3] [ ] [ ]           │
│   8   9  10  ...                        │
└─────────────────────────────────────────┘
```

#### 향후 확장 (필요 시 추가 요청)

- 주간 뷰
- 일간 뷰

## 데이터 구조 (참고)

```typescript
// 습관 정의
interface Habit {
  id: number
  name: string
  levelEasy: string // 1단계 목표 설명
  levelNormal: string // 2단계 목표 설명
  levelChallenge: string // 3단계 목표 설명
  userId: string // 소유 사용자
  categoryId: string
}

// 습관 달성 기록
interface HabitLog {
  id: number
  date: Date
  completedLevel: 1 | 2 | 3
  userId: string // 소유 사용자
  habitId: string
}
```

## UI/UX 참고

- 다른 페이지(Monthly, Weekly, Daily)에서도 습관 관리 기능을 제공할 예정
- 이 페이지(Habits)는 습관 전용 페이지로, 월간 달성 현황만 표시
- 색상/아이콘을 활용해 각 습관을 시각적으로 구분 가능하도록 한다
- 3단계 달성 여부를 직관적으로 구분할 수 있는 시각적 표현 필요 (예: 색상 농도, 숫자, 아이콘 등)
